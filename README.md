## Run project
- Step 1: Git project, run mongodb
- Step 2: Run `npm install` and  `npm install -g @angular/cli` to setup angular/cli
- Step 3: Cd Sever folder run `npm install` and `node sever` to run sever nodejs
- Step 4: Run `ng serve`
